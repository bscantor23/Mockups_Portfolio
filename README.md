## APS (Automatic Payroll System)
---

The **Automatic Payroll System (APS)** is a comprehensive solution designed to automate and streamline payroll processes. It includes features such as:

- Employee data management
- Salary calculation and distribution
- Tax deductions and compliance
- Reporting and analytics

The mockups in the `APS` directory showcase the user interface and workflow of this system.

## Ruppell
---

**Ruppell** is a personal startup project that aims to create multidisciplinary virtual communities and courses under the concept of "nidos" (nests). It is built using **React.js** and **Tailwind CSS** to provide a modern and responsive user experience.
